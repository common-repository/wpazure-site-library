Using Wpazure site library Import pre made websites in just one click.

== Description ==

Wpazure site library is a addon plugin for Wpazure WordPress theme. This plugin contain pre made demos.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wpazure-site-library` directory, or install the plugin through the WordPress Dashboard > Plugins > Add New, Search for "wpazure site library".
2. Install and activate plugin.
3. Now go to the Appearance > Wpazure demo data and import desired site.
